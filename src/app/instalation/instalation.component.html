Requerimentos

Esta é uma biblioteca baseada em Angular8 é necessário que você tenha os seguintes itens instalados no seu computador para prosseguir.

  • NodeJs 10+
  • Angular/CLI 8+

Partiremos nos próximos pontos supondo que você já conhece o framework Angular e já tem seu projeto criado e pronto para a instalação da lib. Se você não conhece o framework Angular acesse o link abaixo para aprender mais sobre.

  • Documentação Angular

Dependencias

A biblioteca possui algumas dependencias e será necessário baixa-las antes de utiliza-la. Abaixo listo as dependencias obrigatórias.

  • PrimeNg
  • Bootstrap
Obs: A lib faz uso de alguns componentes da blibioteca do PrimeNG por isso temos a obrigatóriedade desta dependencia. A dependencia do bootstrap pode não ser obrigatório caso você tenha conhecimento em css pode sobrescrever as classes bootstrap nos componentes da biblioteca.

Icones

Alguns componentes fazem uso de icones, a biblioteca ng-fourward-utils permite a utilização da sua lib de icones favorita ( desde que a mesma seja acessada por classes css ).

Nós recomendamos as seguintes lib de icones:

  • PrimeIcons
  • OU
  • FontAwesome

Com sua biblioteca de icones baixada é preciso configura-la em seu projeto da mesma forma que adicionamos o bootstrap e o primeng
